myFlix Project

Create a REST API with Node.js

Objective:

To build the server-side component of a "movies" web application.

This API holds information about different movies, directors, and genres so that users will be able to search for and learn about their favorite films.

Return data about a director by name /movies/Directors/:Name GET None /movies/Directors/George%Lucas JSON Object containing information about a director Example
